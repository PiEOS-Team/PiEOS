## Build

<i><strong> Note: You may need to install qemu or bochs, gcc, g++ and git before build.</strong></i>

1. clone the repo

```cmd
git clone https://github.com/PiEOS-Team/PiEOS.git
```

2. use `Makefile` to make files

```cmd
chmod +x Makefile
Makefile
```
